# PasiteaVR-WebApp

<h2>1. How to install <h2/>
    <ol>
      <li>
        Clone the repo
      </li>
      <li>
        Run inside the directory "npm install"
      </li>
    </ol>
  <br>
  <h2>2. How to run</h2>
    <ol>
      <li>
        Run in the terminal "node app.js"
      </li>
      <li>
        Go to http://localhost:3000
      </li>
    </ol>
